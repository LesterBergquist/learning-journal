LOgical operators

lOgical operators allow you to compare the results of more than one comparison operator.

((5 < 2) && (2 >= 3))

The expression on the left returns false and so does the expression on the right.

The third expression uses a logical operator to see whether both expressions on either side return true.

In this case they do not, so it evaluates to false.

Loops

oops are conditionals. If the condition is met, a code block will run. Then the condition will be checked

and if it still returns true, the code block will run again, repeating until the condition is not met (is false).

The 3 common types of loops are:

  • "for loops"
  • "while loops"
  • "do while loops"

    • a 'for' loop uses a counter as a condition.

    A while loop returns loops for as long as a condition is true.

    The do...while loop is similar to the while loop but will always run the statement inside the

    curly brackets once, even if the condition evaluates to false.
