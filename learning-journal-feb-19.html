Expressions and Operators

In order for a variable to be useful it needs to be assigned a value.

you can perform operations on the values to determine a single value.

OPerators allow programmers to create a single value from one or more values

There are assignment operators, arithmetic operators and string operators.

The order of execution is the same as in mathematics; so, multiplication before addition for instance.

The string operator '+' joins strings on either side of it.

Functions

Functions let you group a series of statements together to perform a specific task.

You can simply repeat the function rather than repeating all the statements that make it up.
